# AI Evaluator CLI — Go

## Install

```bash
go install gitlab.com/aievaluator/aievaluator-cli/go/cmd/aievaluator@latest
```

## Quickstart

```bash
aievaluator quick "Hello" --expected "Hi!"
aievaluator eval --agent https://my-agent.com/chat --dataset ./tests.json
```
