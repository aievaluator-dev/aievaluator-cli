# AI Evaluator CLI — Node.js

[![npm version](https://img.shields.io/npm/v/aievaluator)](https://www.npmjs.com/package/aievaluator)

## Install

```bash
npm install -g aievaluator
```

## Quickstart

```bash
aievaluator quick "Hello" --expected "Hi!"
aievaluator eval --agent https://my-agent.com/chat --dataset ./tests.json
```
